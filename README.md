# 🏏 IPL Player Performance Dashboard

## 📌 Overview

This project presents an **interactive IPL Player Performance Dashboard** built using Tableau.
It analyzes player and team performance across multiple dimensions such as runs, wickets, strike rate, and country contribution.

The objective of this project is to transform raw cricket data into **clear, actionable insights** through effective data visualization and storytelling.

---

## 🚀 Features

### 🔢 Key KPIs

* **Total Runs:** 1,32,314
* **Total Wickets:** 4,084
* **Total Centuries:** 40
* **Total Fours:** 11,770
* **Total Sixes:** 5,262

---

### 📊 Visualizations

* Top 20 Run Scorers of All Time
* Top 20 Wicket Takers of All Time
* Team-wise Run Distribution (Pie Chart)
* Runs vs Strike Rate (Players with 1000+ runs)
* Country-wise Contribution (Bubble Chart)

---

### 🎛 Interactive Features

* Filters by **Team**
* Filters by **Playing Role**
* Cross-filtering between charts
* Interactive highlighting

---

## 🧠 Key Insights

* Top batsmen consistently maintain a **strike rate between 120–140**
* **Indian players contribute the majority of total IPL runs**
* Players like **Virat Kohli** dominate total runs
* Bowlers like **Dwayne Bravo** lead in total wickets
